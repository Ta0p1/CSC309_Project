# **CSC309 Project**

## **Overview**
Full-stack loyalty program web application for **CSC309**.

- Backend: **Node.js + Express + Prisma**
- Frontend: **React**

---

## **Project Structure**
```
backend/   # Node.js backend (A2 base + extended functionality)
frontend/  # React frontend (to be implemented)
```

---

## **Backend Setup**
Inside `backend/`:

```bash
npm install
npx prisma migrate dev
npm run dev
```

## **Frontend Setup**
Inside `frontend/`:

```bash
npm install
npm start
```

---

## **Notes**
- Backend passes all tests and includes groundwork for extended features.
- Frontend implementation and deployment will be added later.
- This repository will host both backend and frontend for the CSC309 project.
